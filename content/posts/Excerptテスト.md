もしくは、 `excerpt_separator` オプションを削除してください。

```js
{
  resolve: `@tenpamk2/gatsby-theme-figure-blog`,
  options: {
    optionsGatsbyTransformerRemark: {
      excerpt_separator: `<!-- more -->`, // If `undefined` is specified, default pruning method is used.
    },
  }
}
```
